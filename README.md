# 健身计划与饮食计划微信小程序

这是按《健身计划与饮食计划微信小程序设计书》落地的微信原生小程序 MVP 工程。

## 当前实现

- 用户信息分步录入
- 规则引擎生成训练计划
- 规则引擎生成饮食目标和餐次建议
- 训练计划校验提示
- 训练日和动作编辑
- 训练执行页、完成组数记录、休息倒计时
- 饮食食物替换、克数加减与营养重新计算
- 每日打卡
- 体重趋势、训练记录、执行分复盘
- 本地存储模拟用户数据

## V1.1 体验升级

- 首页升级为仪表盘：今日训练、饮食进度、连续打卡和快捷入口。
- 训练页升级为周计划管理：训练日概览、肌群训练量、开始训练入口。
- 新增训练执行页：逐动作完成组数、休息倒计时、训练记录保存。
- 饮食页升级为编辑器：目标热量、三大营养素进度、食物替换、份量加减。
- 数据页升级为复盘页：执行分、完成率、体重趋势条、最近训练记录和建议。

## 训练计划生成流程

- 基础信息优先建档：性别、年龄、身高、体重、目标体重和伤病限制。
- 目标选择放在建档后：减脂、增肌、塑形、保持健康会影响训练结构和饮食目标。
- 训练条件按地点动态展示器械：居家、健身房、户外分别展示不同可选条件。
- 训练频率和时长决定计划规模：每周 2-6 天、每次 15-60 分钟。
- 训练偏好参与规则推荐：系统推荐、分化训练、全身训练、力量加有氧、有氧为主、轻量塑形。
- 饮食习惯同步参与饮食计划：饮食模式、早餐习惯、吃饭场景、忌口食物。
- 生成前先展示推荐方案，用户可以使用推荐方案或查看其他方案。
